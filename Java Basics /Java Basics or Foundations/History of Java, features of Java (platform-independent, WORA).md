## ðŸŸ¢ History of Java

* Java was developed in **1991** at **Sun Microsystems**
* The project was led by **James Gosling**
* Original project name was **Oak**
* Oak was designed for **embedded systems** (TVs, set-top boxes)

### Why Oak failed initially

* Embedded systems market did not grow as expected
* Oak was too heavy for small devices at that time

---

### Birth of Java

* In **1995**, Oak was renamed to **Java**
* Java was redesigned for:

  * Internet
  * Web applications
  * Network-based systems
* First public version released as **Java 1.0**

---

### Growth of Java

* Java became popular due to:

  * Web applets
  * Platform independence
  * Strong security model
* Widely adopted in:

  * Enterprise applications
  * Banking systems
  * Web servers
  * Mobile (Android later)

---

### Java Today

* Sun Microsystems was acquired by **Oracle** in **2010**
* Java is now maintained and released by **Oracle**
* Used in:

  * Backend systems
  * Cloud applications
  * Big data
  * Android development
  * Enterprise software

---

## ðŸŸ¢ Platform Independent

* Java programs are **not dependent on any operating system**
* Java code can run on:

  * Windows
  * Linux
  * macOS
  * Any system with JVM

### How this works

* Java source code (`.java`) is compiled into **bytecode** (`.class`)
* Bytecode is executed by **JVM (Java Virtual Machine)**
* JVM is platform-specific
* Bytecode is platform-independent

---

## ðŸŸ¢ WORA â€“ Write Once, Run Anywhere

* Java follows the principle **WORA**
* Same compiled Java program can run on multiple platforms
* No need to rewrite code for different operating systems

### Example

* Compile Java code once
* Run it on Windows JVM
* Run the same `.class` file on Linux JVM
